## **MOUSE.py**

General logic & utility functions of a Micromouse is abstracted into classes

### 1. `SimulationMouse`

Example code:

```python
mouse = SimulationMouse(algorithm='FLOODFILL')
```

Parameters:

1. algorithm
    - FLOODFILL
    - DFS
    - BFS

### 2. `RealMouse`

Example code:

```python
mouse = RealMouse(algorithm='FLOODFILL')
```

Parameters:

1. algorithm
    - FLOODFILL
    - DFS
    - BFS
